Quick Actions

Getting Started — Connect Stripe

  1. Go to stripe.com → Create a free account
  2. In Stripe Dashboard → Payment Links → Create a link for each product
  3. Copy each Payment Link URL
  4. In Products panel below → Edit each product → Paste the Stripe Payment Link
  5. Your store is ready to sell — no monthly fees, just Stripe's 2.9% + 30¢ per transaction
💡 Pro tip: For the membership product, use a Stripe Subscription Price Link instead of a one-time Payment Link.
